# Learning CSS  

### CSS is a "Cascading Style Sheets" language  

CSS is used for: Content Presentation  

**Syntax**  

```css
selector {
    property1: value;
    property2: value;
    background-color: #FFFFFF;
    color: brown;
}

/* everything selector */
* {
    font-family: Arial;
}

```

**Types of Selectors**
  * Element
  * Class  
  * ID

**Loading CSS**
  * style element in html
  * inline
  * External CSS

```html
<html>
    <head>
        <link rel="stylesheet" href="style.css" />
    </head>
    <body>
        Content goes here...
    </body>
    </html>

```

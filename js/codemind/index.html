Playing the Game

The object of the game is to guess the random four-color code within ten tries.

    A Red peg: - indicates that one of your globes is in the correct position.
    A white peg: - indicates that you have a correct globe, but in an incorrect position.
    A black peg or an empty slot: - indicates an incorrect globe.

NOTE: The pegs are shown in a random order; meaning that the first peg does not correspond to the first globe, so you'll have to figure it out through repetition.

Good Luck!
